# Multi-Broker Portfolio Tracker

Multi-Broker Portfolio Tracker is a Flutter + FastAPI portfolio dashboard that aggregates positions, balances, transactions, and live quotes across LongBridge, IBKR, Futu, and Binance using Firebase-authenticated users and encrypted credential handling (E2E by default, optional server-key mode). For local setup and reproducible end-to-end smoke steps, use [doc/RUNBOOK.md](doc/RUNBOOK.md); for implementation decisions that go beyond the original spec, use [doc/ARCHITECTURE_NOTES.md](doc/ARCHITECTURE_NOTES.md).
